Server Configuration

Manage your Neofrp server configuration.

Configuration Management

Server configuration management is currently handled through the server's JSON configuration file. Future updates will include direct configuration management through this interface.

Example Server Configuration

{
    "log": {
        "log_level": "info"
    },
    "recognized_tokens": [
        "your_secret_token_1",
        "your_secret_token_2"
    ],
    "transport": {
        "protocol": "quic",
        "port": 3400,
        "cert_file": "/path/to/cert.pem",
        "key_file": "/path/to/key.pem"
    },
    "connections": {
        "tcp_ports": [8080, 8081, 8082],
        "udp_ports": [9000, 9001]
    }
}

Configuration Tips

  • Use the templating feature to register port ranges: {{ "{{ expand \"1000-1200\" }}" }}
  • Keep your tokens secure and rotate them regularly
  • Use TLS certificates for production deployments
  • Monitor server logs for connection issues
